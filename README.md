## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# create local database
$ npm run start:db

# create migrations
$ npm run entities:create

# create migrations
$ npm run migration:create --name='migration-name'

# revert last migration
$ npm run migration:down

# revert all migrations
$ npm run migration:revert
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
